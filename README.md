#VisualUIEditor

An UIEditor for Game, base on electron.

![](./screenshot/screenprint.png)

#install
> install node.js first

> npm install -g electron-prebuilt

> npm install -g bower

> bower install

> npm install

#run
open shell in project root and run
> electron .

#start with project
菜单->文件->打开项目(选择目录底下的project-test)即可进行测试
